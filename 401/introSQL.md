## Summary

Learning SQL in this method was quite entertaining and not all too unfamiliar. For the most part, operating on a SQL database feels similar to using an excel worksheet (or perhaps a google doc) albeit through the lens of code. The queries are natural and I felt as though I had more control when searching than I did before with noSQL. SQL databases have these fixed "properties" of each "object" in the form of columns and rows. It translates easily from code, and I'd hope to develop a visual aid similar to what SQL Bolt uses to display a live table of my database if it is not already available.
